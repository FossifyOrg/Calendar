# CalDAV VTODO development

For any CALDAV-NNN task:

1. Read `docs/CALDAV_VTODO_IMPLEMENTATION_ROADMAP.md` completely.
2. Implement only the requested JIRA task.
3. One JIRA task must correspond to one feature branch and one PR into `main`.
4. Do not implement later roadmap tasks in the same PR.
5. Start from an up-to-date clean `main`.
6. Run all checks listed for the task.
7. Do not push or create a PR unless explicitly requested.
